# README

## Steps

### Clone repository

`git clone git@github.com:DanielSantiagoCampos/monokera_soat.git`

### Prepare project to up

- `gem install bundler`
- `yarn install`
- `bundle install`
- `bin/rake db:create`
- `bin/rake db:migrate`
- `rails s`

### Or just go [heroku](https://evening-island-79324.herokuapp.com/)
